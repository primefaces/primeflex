Theming

PrimeFlex has exclusive integration with the Prime UI library theming being used including PrimeFaces, PrimeNG, PrimeReact and PrimeVue by using the colors exported by the theme of the corresponding library. These colors are mainly utilized in background, text and border classes.

Themes

PrimeFlex does not ship with a built-in color scheme by default as it is derived from the Prime UI library. In case you need to use it standalone without a Prime library, include one of the themes from the themes folder. A theme consists of the following set of CSS variables that can be easily customized to build your own.
